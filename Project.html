Task 1

1. Describe in a few words what is the binomial distribution and what are its parameters and their meaning.

a Binomial distribution happens when you do $n$ mutually independent experiments. Each of these experiments have the probability of $p$, and we denote the success of each experiment with $1$, and their failiure with $0$. Notation: $X$ ~ $(n, p)$

2. Plot the binomial distribution with p = 0.7 and size n = 26. Add a suitable title to your plot.

In [ ]:
library(ggplot2)

p <- 0.7
n <- 26
x <- 0:n
y <- dbinom(x, size = n, prob = p)

df <- data.frame(x, y)

# Create the bar plot using ggplot2
ggplot(data = df, aes(x = x, y = y)) +
      geom_col(fill = "#5DA5DA", color = "#252525") +
      labs(title = "Binomial Distribution with p = 0.7 and n = 26",
            x = "Number of Successes",
            y = "Probability") +
      theme_minimal() +
      theme(plot.title = element_text(hjust = 0.5, size = 16, face = "bold"), 
            axis.text = element_text(size = 12),
            axis.title = element_text(size = 14),
            plot.title.position = "plot")

options(repr.plot.width = 18, repr.plot.height = 8)

a Normal distribution (Gaussian distribution) is a continuous probability distribution for a random variable ($X$). It has two parameter: The mean ($$) and the standard deviation ($$).

Mean is the arithmetic mean or average, and standard deviation shows the spread of data.

2. Plot the distribution for $$ = 3 and $$ = 0.4 for $x$ [5, 1]

mu <- -3
sigma <- 0.4
x <- seq(-5, -1, length.out = 100)
y <- dnorm(x, mean = mu, sd = sigma)

ggplot(data.frame(x = x, y = y), aes(x, y)) +
    geom_line(color = "#5DA5DA", size = 2) +
